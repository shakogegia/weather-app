# Weather App

First, install the dependencies:

```bash
yarn install
```

Create `.env.local` file and add your OpenWeatherMap API key:

```bash
EXPO_PUBLIC_API_KEY=your-api-key
```

Then, run the app:

```bash
yarn ios
```
